Instructions for using the Documents Page

Double-click a Folder to access the contents of that folder.

Double-click a File to open the file in the Viewer (PDF) or download the file (non-PDF).

Single-click a File (highlighting it blue) to access the File Menu at the far right.
Clicking the File Menu will bring up the following options (from left to right)

  • Download the File
  • View the File Details

Click the Link Button at the top to paste the URL of the Folder you are currently accessing into the Address Bar of your browser. You can also right-click the Link Button and select “Copy Link Address” or “Copy Shortcut” to paste the URL of the Folder into a document.
