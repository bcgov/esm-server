Instructions for using the Search Page

You can enter additional words into the search box to increase your results. For example, entering memo letter will find documents that contain either of those words. Entering "memo letter" will find documents that contain that exact phrase.

You can narrow your search results by adding a hyphen (minus sign) in front of any word or phrase you wish to exclude from the search. For example, memo -letter will find documents containing the word memo, but not the word letter.

This search feature turns up results based on file names and descriptions. To search the contents of an individual document, you must first view or download it and then perform a search within the document itself.

Double-click a file's name to open the file in the PDF viewer, or download the file (non-PDF and PDF files).

Single-click a file's path to open the file in the documents tab.
